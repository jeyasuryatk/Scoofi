# Scoofi - Goddess of Taste

A modern React application for Scoofi, featuring natural ice cream products, testimonials, and more.

## Technologies Used

- **React**: ^18.3.1
- **Vite**: ^5.4.19
- **TypeScript**: ^5.8.3
- **Tailwind CSS**: ^3.4.17
- **shadcn/ui**: Component primitives
- **Framer Motion**: Animations
- **TanStack Query**: State management

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd gelato-grace-page-main
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

### Development

Start the development server:
```sh
npm run dev
```

### Build

Build for production:
```sh
npm run build
```

Preview production build:
```sh
npm run preview
```

## Features

- **Product Showcase**: Explore our natural ice cream creations.
- **Dynamic Animations**: Smooth transitions and interactive elements using Framer Motion.
- **Responsive Design**: Fully optimized for mobile and desktop devices.
- **Interactive Testimonials**: Real feedback from our happy customers.
